# 🚀 Virtusa Assessment Projects – Java, Python & SQL

This repository contains **three complete projects** demonstrating skills in **Java, Python, and SQL**.
Each project focuses on solving real-world problems with practical implementation.

---

## 📂 Projects Overview

### 1️⃣ 📚 Java Library Management System

A full-stack Java web application to manage library operations like book issuing, returning, and user management.

#### 🔧 Tech Stack

* Java (Servlets, JSP)
* JDBC
* MySQL
* HTML, CSS, JavaScript

#### ✨ Features

* Add / Update / Delete Books
* User Registration & Login
* Issue & Return Books
* Admin Dashboard
* Database Integration

#### ▶️ How to Run

1. Import project into Eclipse/IntelliJ
2. Configure MySQL database using provided schema
3. Run using Apache Tomcat Server

---

### 2️⃣ 💰 Python Smart Expense Tracker

A CLI-based Python application to track daily expenses and analyze spending patterns.

#### 🔧 Tech Stack

* Python
* Pandas (optional)
* CSV / File Handling

#### ✨ Features

* Add daily expenses
* Categorize spending (Food, Travel, Bills)
* Monthly expense summary
* Insights & reports

#### ▶️ How to Run

```bash
python main.py
```

---

### 3️⃣ 🛒 SQL Retail Analysis Project

A data analysis project using SQL to extract insights from retail sales data.

#### 🔧 Tech Stack

* SQL (MySQL / PostgreSQL)
* Excel / CSV Dataset

#### ✨ Features

* Sales Analysis
* Customer Insights
* Revenue Trends
* Product Performance Analysis

#### ▶️ How to Run

1. Import dataset into SQL database
2. Run queries from `.sql` files
3. Analyze results
